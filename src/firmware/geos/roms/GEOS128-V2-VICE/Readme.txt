C128 gebootet vom U36 in Deutsch am VICE

1. Vice C128 starten
2. Unter Settings - Drive ein 1581 Laufwerk mit #8 einrichten
3. ein *.D81 Image in Laufwerk #8 mounten, das Image muß mindestens die Datei "128 desktop" enthalten
4. Unter Settings - C128 settings - Function ROM (Reiter) die Datei "Geos128Ge1581.rom" auswählen
5. Softreste durchführen und nach dem Boot auf den hinteren (schwarzen) 80Zeichen Bildschim klicken.

Boottime bis zum Desktop 15 Sekunden.

Mit diesem Minimalsystem kann man nicht viel machen, daher sollte man auf das *.D81 Image weitere
Apps, Tools und configs speichern. Je mehr Dateien drauf sind, je länger wird jedoch die Boottime,
da alle Dateien eingelesen werden.

Um das System unter VICE bedienen zu können noch die Maus einrichten:

im Vice: 
Settings - Controlport Settings - Port 1 = Mouse 1351
Settings - Mouse settings die Empfindlichkeit etwas heruntersetzte zB auf 25

wenn sich der Windows-Mauszeiger im VICE-Fenster befinden "Alt+Q" drücken
dann steuert man die Maus auf den GEOS-Desktop, mit Alt+Q geht es auch wieder zurück

in anderen VICE-Version ist die Tastemkombination auch "Alt+M"

man kann auch gleich die von mir gesicherten settings im vice einladen
